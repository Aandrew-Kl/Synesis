# Synesis Strategic Advisors Website

Marketing website for Synesis Strategic Advisors, a consulting firm based in Corfu. The site highlights business planning, company valuation, mediation, ESPA funding support, out-of-court settlement services, and tax advisory capabilities.

## Live site

- [esynesis.eu](https://esynesis.eu/)

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React

## Highlights

- Greek-language corporate website with service landing pages
- Static export setup for deployment-friendly builds
- Structured metadata for SEO and social sharing
- Contact-driven user flows for consulting inquiries

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
